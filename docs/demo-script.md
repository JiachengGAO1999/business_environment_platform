# 演示脚本

## 开场（30秒）

> 这是一个面向企业出海的全球营商环境智能分析工作台。它每天汇聚公开资料，筛选出与营商环境相关的政策、经济、贸易和风险信息，并生成可溯源、可审核的国别分析报告。

当前覆盖巴西、沙特阿拉伯、阿联酋三个国家/地区，数据为 Demo 模拟数据。

---

## 1. 工作台首页 `/`（1分钟）

**展示内容**：
- 顶部 5 个 KPI 指标卡：今日新增资料数、高相关资料数、高风险预警数、已生成报告数、待专家审核数
- 下方 3 个国家/地区卡片（巴西、沙特、阿联酋），显示风险等级、机会等级、重点行业和最新摘要
- 右侧面板：今日重点风险 + 今日重点机会

**讲解重点**：
> 第一屏就是可用工作台，不是宣传页。一眼看到全球/区域监测概况。注意风险等级用颜色区分：绿色低风险、黄色中风险、琥珀色高风险、红色严重风险。

**操作**：悬停国家卡片，点击巴西卡片 → 跳转报告详情。

---

## 2. 国别报告 `/reports/`（2分钟）

**操作**：
1. 报告列表页展示 3 份报告（巴西完整报告 + 沙特和阿联酋草稿）
2. 点击巴西报告进入详情

**展示内容（三栏布局）**：
- 左侧：报告目录（10 个章节，含审核状态圆点）
- 中间：报告正文，从摘要开始依次展示宏观经济、产业政策、贸易环境、投资机会、经营风险、政治与地缘风险、企业行动建议、证据来源、待复核问题
- 右侧：证据面板

**关键交互**：
1. 点击左侧目录 → 中间自动滚动到对应章节
2. 点击报告正文中的 `[证:ev-br-001]` 等证据编号 → 右侧面板显示对应来源资料、日期、片段文本、置信度
3. 风险章节有红色/琥珀色标签
4. 底部有"待专家复核问题"琥珀色 callout

**讲解重点**：
> 报告参考了区域国别研究专家的框架，不是普通新闻摘要。每个关键判断都有证据来源。不确定的地方明确标记"需复核"。既有机会分析也有风险分析，最后给企业行动建议。

**操作**：点击右上角"导出报告" → 选择 Markdown → 生成并展示复制功能。

---

## 3. 资料库 `/sources`（1.5分钟）

**操作**：
1. 展示资料表格，注意各列信息密度
2. 筛选"巴西 + 高相关 + 产业政策"
3. 搜索框输入"税制"
4. 点击一条资料行 → 打开详情弹窗

**展示内容（详情弹窗）**：
- 来源机构、国家、类型、可靠性、发布时间
- 摘要
- 命中的分析维度标签
- 证据片段（原文摘录 + 置信度）

**操作**：右上角"导入真实来源" → 展示 URL 模拟抓取 + 手动文本粘贴两个 Tab

**讲解重点**：
> 系统不是简单堆新闻，而是给每条资料打上营商环境分析维度。即使没有真实 API Key，筛选和详情页也完整可用。导入功能展示了后续真实来源接入的数据结构。

---

## 4. 智能筛选 `/screening`（1.5分钟）

**操作**：
1. 展示筛选卡片列表
2. 找一条高相关巴西资料
3. 查看：相关性评分 + **判断理由**（关键！）+ 维度映射标签 + 证据片段
4. 注意"事实摘录"和"模型推断"的视觉分隔

**关键交互**：
- 点击"采纳" → 卡片变绿色 + 顶部反馈提示
- 点击"降权" → 卡片降级
- 点击"标记误判" → 卡片变红色
- 点击"加入今日报告" → 加入生成队列

**讲解重点**：
> 平台不只是一个 AI 分数，而是把模型判断拆成可审核的中间结果：为什么相关、命中了哪个维度、原文证据是什么。专家可以采纳、降权、标记误判。操作状态刷新后保留。

---

## 5. 审核中心 `/review`（1.5分钟）

**操作**：
1. 展示待审核报告列表（左栏）
2. 点击巴西报告 → 右栏展示审核详情
3. 段落审核：找"贸易环境"章节 → 点击"审核此段落" → 选择"需修改" → 输入"对南方共同市场自贸谈判的分析略显乐观" → 提交
4. 段落审核：找"经营风险"章节 → 选择"证据不足" → 提交
5. 整体审核：底部选择"需修改" → 输入整体意见 → 提交

**展示内容**：
- 审核记录历史在底部累计展示
- 返回报告详情页 → 可看到审核意见已体现在报告中

**讲解重点**：
> 平台定位是专家增强工具，不是替代专家。所有 AI 生成内容都留给专家审核入口。审核意见留痕、可追溯。这是区分真正可用的分析工具和玩具的关键。

---

## 6. 收尾（30秒）

**总结**：
> 这个 demo 展示了从资料汇聚、智能筛选、证据抽取、报告生成到专家审核的完整闭环。目前数据是 Mock 的，但数据结构、AgentClient 接口和产品形态都预留了接入真实数据和智能体 API 的空间。

**关键区分点**：
1. 不是营销落地页 — 第一屏就是可用工作台
2. 不是新闻摘要 — 报告遵循专家分析框架
3. 不是黑箱 AI — 每个判断有证据来源
4. 不是替代专家 — 有完整的审核工作流

---

## 演示时长总计：约 8 分钟
